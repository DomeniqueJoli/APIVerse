Sobre o BrainBloom

Um projeto feito para te tirar do loop do doomscrolling e te lembrar que o mundo real vale a pena.

O que é o BrainBloom?

O BrainBloom é uma aplicação web que transforma momentos de tédio em pequenas experiências reais. Em vez de rolar feeds infinitos, você recebe sugestões de atividades para sair do automático.

Como funciona?

Você define preferências como tipo de atividade, custo e número de pessoas. A aplicação usa uma API para gerar sugestões personalizadas em tempo real.

API utilizada

Este projeto usa a Bored API (AppBrewery) , que fornece sugestões dinâmicas de atividades com filtros.
